# 

Todas mudanças realizadas deverão ser documentadas nas entregas seguindo o seguinte formato.

## Caso seja uma mudança na Documentação
``` 
## [dd/mm/aa] hh:mm: breve descrição do que foi feito
#### requisitos adicionados:
#### requisitos removidos:
#### requisitos modificados:
#### justificativas das alterações:
#### impactos das mudanças:
```

## Caso seja uma mudança no Protótipo
``` 
## [dd/mm/aa] hh:mm: breve descrição do que foi feito
#### imagem das mudanças realizada:
#### justificativas das alterações:
#### impactos das mudanças: (declare qual requisito ela impacta)
``` 

## Caso contrário
Isso realmente deveria estar aqui?
